Pulls damage data from config.txt to display estimated boss damage based on your armor.
Net values are based on expert mode damage and defense.

Config file follows the following format. Deviating from this format is likely to break everything
as the error handling is poor.

kingslime=melee,spikedSlimeMelee,spikedSlimeRanged,slimeMelee
eyeofcthulhu=phase1Melee,phase2Melee,sub4%Melee
eaterofworlds=headMelee,bodyMelee,tailMelee
brainofcthulhu=melee,creeperMelee
queenbee=melee,ranged
skeletron=headMelee,limbMelee
wallofflesh=melee,rangedMin,rangedMax,hungryMeleeMin,hungryMeleeMax
